/// Запуск PythonOS

Чтобы запустить PythonOS, нужно установить Python на вашу операционную систему с официального сайта https://www.python.org/ . Либо если вы с андроида, установите Pydroid 3 из Google Play.

/// Установка библиотек для реального браузера

**На андроид** достаточно зайти в гамбургер-меню (три полоски слева сверху), нажать "Pip" и вписать "requests beautifulsoup4 html2text".

**На Windows** делаем следующее:
Заходим в проводник, находим папку с "PythonOS" или "os", в адресной строке проводника убираем все заменяя на "cmd" и нажимаем "Enter". После этого пишем следующие команды:
py --version (проверка версии, должна быть 3.13)
py -m pip install requests
py -m pip install beautifulsoup4
py -m pip install html2text

Запускаем проект.
P.S. на Windows будут иероглифы вместо текста в браузере, пока не знаю как фиксить
